OneDrive

Enterprise File Sync and Share

Features

  • Cross Platform File Access: OneDrive gives users the ability to access files from any device at any time. Real-time sync will upload documents from local devices to the cloud.
  • Offline Access: Sync capabilities allow files to be stored in the OneDrive cloud and synced to a device for fast file access even while offline.
  • Collaboration: Share files with anyone. Collaborate in real-time across the world with colleagues inside and out of your organization.

Latest Updates

Launched

Shared by me view (OneDrive web UI only)

Q1 CY2018

OneDrive web users will use this new view to see the files in their OneDrive that are shared with others.

New office desktop sharing experience (Windows)

Q1 CY2018

OneDrive has a universal sharing UI both in web view and in File explorer / Mac finder so that users have a seamless experience. This sharing UI has now also been standardized across Office desktop clients of Word, PowerPoint and Excel.

Modern Web experience for Office files

Q1 CY2018

We're introducing a faster, more collaborative Word experience for SharePoint and OneDrive documents on the web. You'll be able to get started right away on typing and collaborating now that documents open directly into an Edit mode.

IRM library sync support for Windows

Q1 CY2018

OneDrive sync client (NGSC) for Windows will support syncing of IRM (Information Rights Management) protected libraries in SharePoint Online.

OneDrive for Business multi-geo and service level encryption support (private preview only)

Q1 CY2018

With O365 multigeo, organizations would be able to save their OneDrive and Sharepoint data in their respective regions in order to abide by their security and legal rules. With customer key, service level encryption will be made available by allowing customers to hold their own encryption key - More Information

Rolling Out

Customized Sharing Defaults per SharePoint Site Collection

May CY2018

In SharePoint Online, Default sharing link types and Default sharing link permissions are now available to be customized per site collection. In SharePoint Online, Default sharing link types and Default sharing link permissions are now available to be customized per site collection. You can now configure the default link type and default link permission for your users in SharePoint Online and OneDrive for Business. With this change, you may now further customize the default sharing settings on a per site collection basis.

Integration of Sync client into Mac Office

May CY2018

The OneDrive client will now be installed as part of the Office for Mac install process.

File viewers in Teams

May CY2018

OneDrive File viewers extended to Teams

Versioning settings in OneDrive for Business, Groups and team sites in SharePoint Online

May CY2018

All SharePoint and OneDrive libraries will be set to retain a minimum of one hundred major versions. Existing libraries that have versioning enabled but are set to retain fewer than one hundred major versions will be updated to retain the new minimum. Libraries already set to retain one hundred or more major versions will not be affected, including those with the default setting of five hundred. With these changes, the Document Library Settings page will no longer support the ability to disable versioning or configure it to retain fewer than one hundred versions.

File Hover Card

Q1 CY2018

With the new OneDrive Web UI refresh, we have launched another feature called the File Hover card. As soon as you hover your mouse over a file you will be able to see file details like access stats - number of views, who viewed , who modified.
